Emaar Properties Stock Performance Dashboard

Power BI Finance Analysis Project (2021–2026)

Project Overview

This project analyzes the historical stock performance of Emaar Properties using Power BI. The dashboard evaluates price trends, trading volume behavior, and the relationship between price movement and investor activity from 2021 to 2026.

Key Insights

  • Stock price increased by approximately 170% from 2021 to 2026.
  • Trading volume grew more than 3×, indicating strong investor confidence.
  • Positive price–volume correlation suggests sustainable market growth.

Tools & Skills Used

  • Power BI Dashboard Design
  • Data Cleaning & Transformation
  • Financial Trend Analysis
  • Data Visualization & KPI Reporting
